# Comments in Go

## Single-line Comments

```go
// This is a single-line comment
```

## Multi-line Comments

```go
/*
This is
a multiline
comment
*/
```

## Doc Comments

Doc comments are used to provide details and descriptions of a program

```go
// File Name   : hello.go
// Description : This program prints hello world
// Author      : Jane Doe
// Date        : 1234.12.25
```
