# Booleans in Go

## Booleans in Go

Booleans are a [basic](go_data-types_basic.md#booleans) [value](go_data-types_value-vs-reference.md#value-types) type in Go

| Data Type | Description       |
| --------- | ----------------- |
| `bool`    | `true` or `false` |

## Zero Value for Boolean Types

> Zero value for a boolean is `false`
