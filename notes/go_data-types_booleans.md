# Booleans in Go

## Booleans in Go

Booleans are one of the [basic types](go_data-types_basic.md#booleans) in Go

| Data Type | Description       |
| --------- | ----------------- |
| `bool`    | `true` or `false` |

## Zero Value for Boolean Types

> Zero value for a boolean is `false`
